# **App Name**: ExamPaper AI

## Core Features:

- Board-Aligned Exam Paper Generation: AI-powered engine generates complete exam papers by selecting State, Board, Class, Subject, Chapters, and Total Marks. Generates syllabus-matched, difficulty-balanced papers that follow official board patterns.
- Automated Exam Layout: Automatically formats the generated questions into printable exam layout, complete with section-wise structure (A/B/C/D), correct mark distribution, and clear instructions.
- PDF Export & Multi-Set Generation: Provides instant PDF export for generated papers. It can create multiple paper sets (Set A/B/C).
- AI-Powered Question Adaptation and Regeneration Tool: Regenerate individual questions on demand, adapting to different board styles, preventing repeated question patterns, and balancing difficulty using AI as a tool.
- Language Translation: Translate exam papers between English, Hindi, Gujarati, and other Indian languages using AI.
- Teacher Content Upload and OCR Question Conversion: Teachers can type their own questions, upload PDF/Word files for AI extraction, or scan textbook pages for OCR conversion into questions.
- Firebase Integration: Integrate with Firebase Auth for Google Sign-In, Firestore for data storage, and Storage for file management and PDFs. The backend logic lives inside Cloud Functions.

## Style Guidelines:

- Primary color: Saturated blue (#2979FF), evoking trust, intelligence, and reliability, aligning with the app's function in education.
- Background color: Light blue (#E3F2FD), creating a calm and professional environment for teachers to work in.
- Accent color: Purple (#7C4DFF), for a distinct contrast used to highlight key interactive elements.
- Body and headline font: 'PT Sans' for a clean and readable interface.
- Simple, clear icons for ease of use, adhering to Material Design principles.
- Clean and intuitive layout, optimized for speed and ease of use on both mobile and web platforms. Follow the prescribed 4-screen limit: Login, Generate Paper Form, Paper Preview + Edit, Export + History
- Subtle animations for user feedback and engagement.
